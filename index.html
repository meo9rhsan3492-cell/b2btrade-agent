使用方式

3步开启 AI 外贸

01

安装

一行命令安装，无需配置，直接使用。

npm i -g b2btrade-agent
02

配置 API Key

支持 OpenAI / Claude / MiniMax，输入你的 Key 即可。

b2b config
03

发出指令

用自然语言描述需求，AI 自动执行。

b2b chat
