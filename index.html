-- Quick Start

3步开始

安装配置只需2分钟，之后随时调用。

01

Install

需要Node.js 18+，一行命令安装。

npm install -g b2btrade-agent
02

Configure

首次运行按提示输入API Key，全程引导。

b2b config
03

Run

用自然语言描述你的需求，AI自动执行。

b2b chat
