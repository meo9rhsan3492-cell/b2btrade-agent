-- Quick Start

3 Steps to Start

Install and configure in 2 minutes, then use anytime.

01

Install

Requires Node.js 18+, one command to install.

npm install -g b2btrade-agent
02

Configure

First run prompts for API Key. Supports Claude/GPT/MiniMax.

b2b config
03

Run

Describe your needs in natural language, AI executes.

b2b chat
