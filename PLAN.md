# b2btrade-agent 产品规划

> 目标用户：外贸从业者（不是开发者）
> 核心价值：AI 把外贸获客/跟进/报价的重复工作自动化

---

## 一、定位问题（现在最大的坑）

**当前定位混乱：**
- 技术文档太多，看不懂
- 没有 Demo，看不到效果
- 门槛太高，需要 API Key 才能体验

**重新定位一句话：**
> "外贸人的 AI 助手，1 分钟找到客户，3 分钟生成开发信"

---

## 二、版本路线图

### v1.1.x —— 降低门槛（当前优先级最高）

| 任务 | 收益 | 工作量 |
|------|------|--------|
| 加演示模式（`b2b demo` 不需要 API Key） | 用户零成本体验，留存率↑ | 1天 |
| 做 1 个 GIF 演示图 | 放到 README 首屏 | 2小时 |
| 发布 npm 包 | 开发者精准流量 | 1小时 |
| 提交 ClawHub/SkillHub | 平台精准流量 | 1小时 |

### v1.2.x —— 强化核心（1-2周）

| 任务 | 收益 |
|------|------|
| CRM 数据 + 微信绑定（Email通知） | 真实使用场景 |
| 邮件发送集成（Gmail SMTP） | 完整闭环 |
| 展会名单批量导入（Excel） | 展会场景 |
| 报价单 PDF 导出 | 真实出单 |

### v1.3.x —— 增长（3-4周）

| 任务 | 收益 |
|------|------|
| 小程序/H5 轻量版（不用装 Node.js） | 用户量爆发 |
| Chrome 插件（选中文本→右键分析） | 场景渗透 |
| 多语言（英语/阿拉伯语/西班牙语） | 中东/南美市场 |
